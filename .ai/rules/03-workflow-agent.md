# 03 - Agent workflow

## Before modification

- Read `AGENTS.md` and `.ai/rules/*.md`.
- Briefly summarize applied constraints.

## During modification

- Make minimal and coherent changes.
- Preserve the existing style.
- Do not mix multiple technical topics in the same pass unless necessary.

## After modification

- Verify `make phpstan`.
- Verify `make fix-check`.
- Do not commit without explicit request.
